# arduino_home

A home system consisting of ESP8266 boards which:

- Connect to a led strip and control the brightness
- Connect to a ws2812b 8 led ring and control the brightness and the color.
- Act as a control panel interface which shows all the devices, their statuses. Feature are mentioned below

 
There is a common UI language among all interface with design components that control the number of LED that are open, or the brightness or the color and the on/off state of the device. There is also a timer to set the light brightness in a set duration.
